# Tictactoe Games
You can do online test in my Heroku [here](http://reddit-clone-demo.herokuapp.com/)

### Installation

```sh
$ git clone 'repo-name'
$ cd 'repo-name'
$ npm install or yarn install
$ npm run start or yarn start
```
